# <img alt="EveryType logo" height="64" src="logo.svg" width="64"/> EveryType

The EveryType library provides facilities for obtaining references to all .NET
types that satisfy a given predicate.

## Installation

The EveryType library can be installed using
[NuGet](https://www.nuget.org/packages/EveryType/).

## Usage

API documentation for the EveryType library may be found
[here](https://jeffrey-w.github.io/EveryType/).

## Contributing

The EveryType library is privately maintained. Please open an issue to request
a feature or report a bug. If you would like to contribute, you may request an
invitation to collaborate on this repository.
