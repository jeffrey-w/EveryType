# EveryType

The EveryType library provides facilities for obtaining references to all .NET
types that satisfy a given predicate.

## Installation

The EveryType library can be installed using
[NuGet](https://www.nuget.org/packages/Extra.Extensions/).

## Code Repository

Source code for EveryType may be found
[here](https://github.com/jeffrey-w/EveryType).
